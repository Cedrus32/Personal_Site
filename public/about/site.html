    • Accessibility

    I've tried to make this site accessible for disability and mobile users. Below are current and future accessibility features:

    Current Features

    • Mobile-friendly layout.
    • 200-zoom compatible.
    • Keyboard navigable.
    • "Skip to content" button available to keyboard and screen reader users.
    • Structured with semantic HTML.
    • Toggleable light/dark theme and gif animations.
    • Light/dark theme and gif animations initially match user preferences.
    • Light/dark themes have sufficient contrast (WCAG AA support) for normal and large text.
    • Alternative font selection, including readable and dyslexic-friendly fonts.
    • All text links are visually identified with an underline.
    • External and pop-up text links are visually identified with an arrow.
    • Lazy load images.
    • Descriptive alt text for all images.
    • JS optional. (note: Some features require JS, including lazy image loading and theme/animation/font toggles.)

    Future Features

    • Convert growth charts to tables for tabability.
    • Minify HTML, JS, and CSS.
